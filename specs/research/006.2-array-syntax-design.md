# Story 006.2: Array Syntax Design

**Status**: In Progress  
**Date**: 2026-05-20

## Goal

Design comprehensive LULCODE array syntax that:
1. Feels natural in LULCODE context
2. Transpiles cleanly to string-encoded BUKKIT
3. Supports self-hosting requirements (transpiler needs)
4. Maintains LOLCODE whimsy while being practical

## Design Principles

1. **LOLCODE Spirit**: Maintain verbose, silly keywords
2. **Modern Ergonomics**: Support clean bracket notation
3. **Backward Compatible**: Pure LOLCODE still works
4. **Self-Hosting First**: Optimize for transpiler use cases

## Array Creation

### Option A: Bracket Literal (Modern)

```lulcode
VAR empty ITZ []
VAR nums ITZ [1, 2, 3]
VAR mixed ITZ ["hello", 42, TROOF]
```

**Pros**: Familiar, concise, clear
**Cons**: Less "LOLCODE-like"

### Option B: LOLCODE Verbose

```lulcode
I HAS A empty ITZ AN ARRAY
I HAS A nums ITZ AN ARRAY WIF 1 AN 2 AN 3
```

**Pros**: More LOLCODEish
**Cons**: Verbose, harder to parse

### **Recommendation**: **Support Both**

```lulcode
VAR arr ITZ []                    BTW Modern style
VAR arr ITZ AN ARRAY              BTW LOLCODE style (equivalent)
VAR nums ITZ [1, 2, 3]            BTW Literal with values
```

**Transpilation**:
```lolcode
I HAS A arr ITZ A BUKKIT
arr HAS A __length ITZ 0
arr HAS A __is_array ITZ WIN

BTW For [1, 2, 3]:
arr HAS A __length ITZ 3
arr HAS A __0 ITZ 1
arr HAS A __1 ITZ 2
arr HAS A __2 ITZ 3
```

## Array Access

### Read Access

```lulcode
VAR x ITZ arr[0]                  BTW First element
VAR y ITZ arr[i]                  BTW Variable index
VAR z ITZ arr[-1]                 BTW Last element (Python-style)
```

**Transpilation**:
```lolcode
BTW arr[0]
I HAS A x ITZ arr'Z __0

BTW arr[i] - requires runtime computation
I HAS A __key ITZ SMOOSH "__" AN i MKAY
I HAS A y ITZ arr'Z SRS __key

BTW arr[-1] - negative index
I HAS A __idx ITZ DIFF OF arr'Z __length AN 1
I HAS A __key ITZ SMOOSH "__" AN __idx MKAY
I HAS A z ITZ arr'Z SRS __key
```

### Write Access

```lulcode
arr[0] = "hello"                  BTW Literal index
arr[i] = value                    BTW Variable index
arr[arr[0]] = 42                  BTW Nested access
```

**Transpilation**:
```lolcode
BTW arr[0] = "hello"
arr'Z __0 R "hello"

BTW arr[i] = value - runtime computation
I HAS A __key ITZ SMOOSH "__" AN i MKAY
arr'Z SRS __key R value
```

## Array Length

**Reuse existing LENGZ OF**:

```lulcode
VAR len ITZ LENGZ OF arr
```

**Transpilation**:
```lolcode
BTW Return __length metadata
I HAS A len ITZ arr'Z __length
```

**Alternative**: Generate runtime function:
```lolcode
HOW IZ I __LULCODE_ARRAY_LENGTH YR arr
  FOUND YR arr'Z __length
IF U SAY SO

I HAS A len ITZ I IZ __LULCODE_ARRAY_LENGTH YR arr MKAY
```

## Array Operations

### PUSH (Add to End)

```lulcode
PUSH "value" TO arr               BTW LOLCODE style
arr.push("value")                 BTW JavaScript style (optional)
```

**Transpilation**:
```lolcode
BTW Runtime function
HOW IZ I __LULCODE_ARRAY_PUSH YR arr AN YR value
  I HAS A len ITZ arr'Z __length
  I HAS A key ITZ SMOOSH "__" AN len MKAY
  arr'Z SRS key R value
  arr'Z __length R SUM OF len AN 1
IF U SAY SO

I IZ __LULCODE_ARRAY_PUSH YR arr AN YR "value" MKAY
```

### POP (Remove from End)

```lulcode
VAR last ITZ POP FROM arr         BTW Returns last element
```

**Transpilation**:
```lolcode
HOW IZ I __LULCODE_ARRAY_POP YR arr
  I HAS A len ITZ arr'Z __length
  BOTH SAEM len AN 0, O RLY?
    YA RLY
      FOUND YR NOOB
  OIC
  
  I HAS A idx ITZ DIFF OF len AN 1
  I HAS A key ITZ SMOOSH "__" AN idx MKAY
  I HAS A value ITZ arr'Z SRS key
  
  BTW Delete key (set to NOOB)
  arr'Z SRS key R NOOB
  arr'Z __length R idx
  
  FOUND YR value
IF U SAY SO
```

### SHIFT (Remove from Start)

```lulcode
VAR first ITZ SHIFT FROM arr      BTW Returns first element
```

**Transpilation**:
```lolcode
HOW IZ I __LULCODE_ARRAY_SHIFT YR arr
  I HAS A len ITZ arr'Z __length
  BOTH SAEM len AN 0, O RLY?
    YA RLY
      FOUND YR NOOB
  OIC
  
  BTW Get first element
  I HAS A value ITZ arr'Z __0
  
  BTW Shift all elements down
  I HAS A i ITZ 0
  IM IN YR shift_loop UPPIN YR i TIL BOTH SAEM i AN DIFF OF len AN 1
    I HAS A next ITZ SUM OF i AN 1
    I HAS A curr_key ITZ SMOOSH "__" AN i MKAY
    I HAS A next_key ITZ SMOOSH "__" AN next MKAY
    arr'Z SRS curr_key R arr'Z SRS next_key
  IM OUTTA YR shift_loop
  
  BTW Remove last element
  I HAS A last_idx ITZ DIFF OF len AN 1
  I HAS A last_key ITZ SMOOSH "__" AN last_idx MKAY
  arr'Z SRS last_key R NOOB
  arr'Z __length R last_idx
  
  FOUND YR value
IF U SAY SO
```

### UNSHIFT (Add to Start)

```lulcode
UNSHIFT "value" TO arr            BTW Add to beginning
```

(Similar transpilation to SHIFT, but in reverse)

## Array Slicing

**Reuse string slice syntax**:

```lulcode
VAR subset ITZ arr[0:3]           BTW Elements 0, 1, 2
VAR last5 ITZ arr[-5:]            BTW Last 5 elements
VAR copy ITZ arr[:]               BTW Full copy
```

**Transpilation**: Generate slice function similar to string slice

## Array Iteration

### FOREACH with Arrays

```lulcode
FOREACH element IN arr
  VISIBLE element
END
```

**Transpilation**:
```lolcode
I HAS A __idx ITZ 0
I HAS A __len ITZ arr'Z __length
IM IN YR foreach_loop UPPIN YR __idx TIL BOTH SAEM __idx AN __len
  I HAS A __key ITZ SMOOSH "__" AN __idx MKAY
  I HAS A element ITZ arr'Z SRS __key
  VISIBLE element
IM OUTTA YR foreach_loop
```

### FOREACH with Index

```lulcode
FOREACH i, element IN arr
  VISIBLE "Index {i}: {element}"
END
```

**Transpilation**: Similar, but expose loop counter as `i`

### Manual FOR Loop (Also Works)

```lulcode
FOR i FROM 0 TO LENGZ OF arr
  VISIBLE arr[i]
END
```

## Type Checking

### Distinguish Arrays from BUKKIT

**Internal marker**: `__is_array` metadata key

```lulcode
IF arr IS AN ARRAY
  BTW Array-specific logic
END
```

**Transpilation**:
```lolcode
arr'Z __is_array, O RLY?
  YA RLY
    BTW Array-specific logic
OIC
```

## Edge Cases

### 1. Sparse Arrays
```lulcode
VAR arr ITZ []
arr[0] = "a"
arr[5] = "b"
BTW arr[1] through arr[4] are NOOB
```

**Solution**: Allow gaps, return NOOB for missing indices

### 2. Negative Indices
```lulcode
arr[-1]    BTW Last element
arr[-2]    BTW Second to last
```

**Transpilation**: Convert to positive index at runtime

### 3. Out of Bounds
```lulcode
arr[100]   BTW When length is 5
```

**Solution**: Return NOOB (like JavaScript returns undefined)

### 4. Mixed Array/Dictionary Usage
```lulcode
arr[0] = "array value"
arr["name"] = "dict value"
```

**Solution**: Allow but discourage (document as anti-pattern)

### 5. Nested Arrays
```lulcode
VAR matrix ITZ [[1, 2], [3, 4]]
VISIBLE matrix[0][1]    BTW Should be 2
```

**Solution**: Support via recursive transpilation
